# Root object fields

The FastStore API provides access to additional data fields that are not included in the [native FastStore GraphQL schema](/reference/api/queries). These fields can be accessed through the root object of each query. You can find more information on the available fields below.

:::warning
Keep in mind that the fields below may not be available in the [native FastStore GraphQL schema](/reference/api/queries). To use the ones that are not available in the native schema, you must [extend the schema](/how-to-guides/faststore-api/extending-the-faststore-api). Learn more about how you can [use the root object fields in yout storefront](/how-to-guides-faststore-api/using-the-faststore-api-root-object-fields).
:::
